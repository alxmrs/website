What’s this website all about?

It’s a place for me to cross reference public profiles, host assets like profile pics, and occasionally share my thoughts (usually on programming).

Setup

Pre-requisite: Install pandoc - I installed pandoc via Anaconda: 1. Install Anaconda 1. Create a conda environment for working with this website: conda create -n website python=3.6 -y 1. conda activate website 1. conda install pandoc 1. Add, remove, edit Markdown files 1. ./build.sh 1. Serve the static contents to see if you’re happy with the site. 1. Commit + push the changes to master 1. That’s it!
